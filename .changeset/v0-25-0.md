v0.25.0 — All channels. All memories. All models.

Inbox: `@tuttiai/inbox` orchestrator routes inbound messages from five platforms (Telegram, Slack, Discord, Email, WhatsApp) into score-defined agents. Allow-lists, per-user token-bucket rate limiting, per-chat serial queues with bounded depth, typed `inbox:*` events. Each platform stays in its voice (`@tuttiai/telegram`, `@tuttiai/email`, `@tuttiai/whatsapp` are new this release; `@tuttiai/slack`, `@tuttiai/discord`, `@tuttiai/postgres`, `@tuttiai/mcp` updated) and inbox dispatches via the voices' shared client wrappers so a single token only opens one platform connection.

Curated memory + user model: The agent itself calls `remember` / `recall` / `forget` (alongside the existing system-prompt injection) when `agent.memory.semantic.curated_tools !== false`. Both surfaces share one `MemoryEnforcer` so caps, true-LRU eviction, and `memory:*` events fire exactly once per logical operation. New dialectic `UserProfile` doc per `user_id` (`summary`, `preferences`, `ongoing_projects`) consolidated by an LLM pass every N turns from `UserMemoryStore` — sits alongside `user_memory`, doesn't replace it.

OpenRouter: New `OpenRouterProvider` exposes 300+ models across providers via one API key. Reuses the OpenAI SDK with `baseURL = https://openrouter.ai/api/v1`. Cost surfaces inline on `usage.cost_usd` (no second `/generation` round trip).

Migration: fully additive. Single rename: `AgentConfig.semantic_memory` → `AgentConfig.memory.semantic`.
